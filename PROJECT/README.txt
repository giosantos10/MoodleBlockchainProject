CS 189 Project: Moodle Blockchain
by Stephen John M. Ramos & Angio Gabriel D. Santos

Requirements
1. Install node-js on your computer

Setting up the application
1. Extract the PROJECT folder from the ZIP file: Ramos_Santos_Project.zip
2. Open the terminal and navigate to the directory of the PROJECT folder you just extracted.
3. Type: node server.js     and then press Enter.
4. Go to your browser and go to the URL where the application is running in: http://127.0.0.1:8129/
